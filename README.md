# vue-koa-tolist
简单的vue-koa的代办事
## 使用说明：
- `node` >= 8.10.0 数据库为`mysql`
- 在`server`文件夹里面`npm install`
- 在`server/config/db.js`将密码改成你数据库的密码
- 打开数据库，建立数据库todolist,然后在server文件夹里运行`node app.js`,就会在数据库todolist里面建立`user`表和`list`表，然后在user表建立一个用户密码一定
